Overview

Recipe Finder is a responsive web application that allows users to search and explore recipes through an intuitive interface. The application fetches recipe data from an external API and displays relevant recipe information dynamically.

Features
Search recipes by name
View recipe details instantly
Dynamic data fetching using API
Responsive and user-friendly interface
Tech Stack
HTML
CSS
JavaScript
REST API
Project Structure
index.html – Application structure
style.css – Styling and layout
script.js – API integration and functionality
How to Use
Open the application in a web browser.
Enter a recipe name in the search bar.
Browse the displayed recipes and explore details.
Learning Outcomes
API integration using JavaScript
Dynamic content rendering
DOM manipulation
Responsive web design
